# 如何训练你的 LLM

一个可交互的探索式解释页面，通过亲手操作的游戏化机制，讲解大语言模型是如何被训练出来的——从原始文本一路走到对齐后的助手。

[English](./README.md) · **中文**

**[在线体验 →](https://airingursb.github.io/how-to-train-your-llm)**

## 你会学到什么

游戏覆盖了完整的 LLM 训练流程，分为 3 幕、10 章：

**第一幕：预训练**
- 第 0 章：魔术表演 —— 观察逐 token 的文本生成
- 第 1 章：分词 —— 通过合并字符对，亲手造一个 BPE 分词器
- 第 2 章：猜词游戏 —— 和 bigram 模型比赛预测下一个词
- 第 3 章：注意力 —— 画出注意力连接，探索 Q/K/V
- 第 4 章：Transformer —— 用拖拽方式拼装出整个架构
- 第 5 章：喂养巨兽 —— 实时调控训练超参数

**第二幕：监督微调（SFT）**
- 第 6 章：听话的学生 —— 在 A/B 对比中认出 SFT 模型
- 第 7 章：以例施教 —— 以数据标注者的身份为回复排序

**第三幕：RLHF 与对齐**
- 第 8 章：人类想要什么？ —— 从偏好数据训练一个奖励模型
- 第 9 章：对齐游戏 —— 在有用、无害、诚实之间寻找平衡

## 技术栈

- **渲染：** Canvas 2D + DOM 混合的分屏布局
- **动画：** GSAP、CSS 动画、requestAnimationFrame
- **音频：** Web Audio API（合成的环境音乐 + 音效）
- **手绘风格：** Rough.js
- **构建：** Vite
- **国际化：** 中英双语，支持热切换语言
